# Polysemy Learning Norming

Standalone browser app for the noun-to-verb low-relatedness norming round used in the Polysemy Learning project.

## Files

- `noun_verb_norming.html`: single-page app
- `noun_verb_norming.js`: embedded candidate set, randomization, local persistence, and CSV/JSON export

## Usage

Open `noun_verb_norming.html` in a browser.

- `24件`: full prepared round-1 set
- `10件`: shorter reviewed subset

Responses stay in the browser and can be exported as:

- long CSV
- wide CSV
- JSON

No server is required.
